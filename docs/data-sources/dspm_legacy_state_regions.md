# visionone_dspm_legacy_state_regions (Data Source)

Discovers the GCP regions a legacy Terraform Package Solution deployment used, by reading `gs://trendmicro-v1-{project_id}/default.tfstate`. Returns an empty set when no legacy bucket or state file exists. Pair with `visionone_dspm_legacy_cleanup_region` to drive per-region cleanup before redeploying via the TFP path.

## Use Cases

- **Region Discovery**: Enumerate the GCP regions a legacy Terraform Package Solution actually deployed into, by parsing `gs://trendmicro-v1-{project_id}/default.tfstate`.
- **Migration Driver**: Pair with `visionone_dspm_legacy_cleanup_region` (typically via `setunion` with the new TFP location list) to drive per-region cleanup before redeploying via the Terraform Provider path.
- **Safe on Fresh Installs**: Returns an empty set when no legacy bucket or state file exists, so downstream `for_each` cleanup becomes a no-op.

## Example Usage

```terraform
# Discover which GCP regions the legacy Terraform Package Solution was running
# in for a given project. Returns an empty set when no legacy state exists,
# letting downstream cleanup resources be a no-op for fresh installs.
data "visionone_dspm_legacy_state_regions" "example" {
  project_id          = "my-gcp-project-id"
  service_account_key = var.legacy_service_account_key
}

output "legacy_regions" {
  value = data.visionone_dspm_legacy_state_regions.example.regions
}
```

## Schema

### Required

- `project_id` (String) The GCP project ID whose legacy state bucket should be inspected.

### Optional

- `service_account_key` (String, Sensitive) Base64-encoded JSON service account key used to authenticate with GCS for reading the legacy state file. Optional — three common patterns:

- **CAM-integrated** (recommended): set to `visionone_cam_service_account_integration.comprehensive.private_key`.
- **BYO key**: set to a base64-encoded JSON key for any service account with `storage.objects.get` on the legacy bucket.
- **ADC**: omit the attribute entirely. Falls back to Application Default Credentials.

### Read-Only

- `bucket_name` (String) The legacy state bucket name that was probed (`trendmicro-v1-{project_id}`).
- `id` (String) Equals `project_id`.
- `regions` (Set of String) Set of GCP region names extracted from the legacy state file. Empty when no legacy state exists.

## Required Permissions

The authenticating principal must be able to read the legacy state bucket:

- `storage.buckets.get`
- `storage.objects.get`
